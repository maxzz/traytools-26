### `PanelSizes` type (`2-panel-sizes.ts`)

Changed from fixed `horizontal` / `vertical` fields to a named map:

```typescript
export type PanelSizes = Record<string, Layout>;
```

Group names are defined in `PANEL_GROUPS`:

| Group key | Default layout |
|---|---|
| `demos.resizable.horizontal` | `{ left: 30, right: 70 }` |
| `demos.resizable.vertical` | `{ top: 50, bottom: 50 }` |
| `trace-manager.main` | `{ panels: 68, categories: 32 }` |
| `trace-manager.left` | `{ list: 38, view: 62 }` |

`getValidPanelSizes` merges stored values with these defaults and migrates the old `{ horizontal, vertical }` shape automatically.
