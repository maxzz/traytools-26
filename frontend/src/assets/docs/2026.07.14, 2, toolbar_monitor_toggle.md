DPAgent toolbar is ported from the legacy traytools app.

### Behavior
- **View → DPAgent toolbar** toggles the header strip and the 1s run monitor together
- Hidden toolbar → monitor stops (component unmounts, interval cleared)
- Visibility persisted as `appSettings.showDpAgentToolbar` (default `true`)
- **View → DPAgent run options** has Start-as-High + unload hook (same as the old submenu)

### Toolbar
`[icon] [Start] [Stop] [AG integrity] | [Exit] [TT integrity]`

- Start / Stop enable based on running state  
- Stop closes the agent window and broadcasts unload  
- Integrity badges: `?` / `H` / `M` / `M+` / `L`

### Key files
- Backend: `backend/dpagent/` (FindWindow status, start/stop, integrity)
- UI: `frontend/src/components/1-header/4-dpagent-toolbar/`
- Settings: `showDpAgentToolbar`, `startDpAgentHigh` in `1-ui-settings.ts`
