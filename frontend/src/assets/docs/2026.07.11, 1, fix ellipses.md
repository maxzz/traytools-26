---

**Cursor**

`truncate` conflicts with `break-all`, and ellipsis won't work in a flex row with multiple children unless the cell can shrink (`min-w-0`) and text stays on one line.


Three things were blocking the ellipsis:

1. **`break-all` vs `truncate`** — `truncate` needs `white-space: nowrap` (single line). `break-all` allows wrapping, so they cancel each other out.

2. **`flex` with multiple children** — `text-overflow: ellipsis` only works on one line of text in a single box. With `flex` + `gap-3`, each span is its own flex item, so the parent never ellipsizes as one string.

3. **Missing `min-w-0`** — In a grid column (`1fr`), the default `min-width: auto` stops the cell from shrinking below its content, so overflow never triggers.

**Fix:** Use `min-w-0 truncate` on the cell, drop `break-all` and `flex`, and keep the colored parts as inline `<span>`s with spaces between them so they truncate together.

Applied the same `min-w-0 truncate` fix to the Thread row, which had the same `break-all truncate` conflict.
